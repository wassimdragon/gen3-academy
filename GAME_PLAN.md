# 🎮 Gen 3 Academy — Game Plan (v1)

Brainstormed design for the game/interactive version. Fully compliant with the project docs
(earned knowledge, guide-not-answer AI, no depictions of living beings, no gambling mechanics).

## Flow (open → play)
- **New player:** Create emblem + name → quick **"welcome / how it works"** intro → map
- **Returning player:** straight to the **map** where they left off
- **Finish a region:** **celebration + reward** moment (badge and/or new decoration unlocked)

## Start
- Pick & customize an **emblem** (lantern / crest / calligraphy) + name
- Emblem is a **non-living** design (no faces/animals)

## Map
- Each **lesson = a region**
- Stages along a **path (1 → 2 → 3 ...)**, unlocked **strictly in order**
- Locked regions/stages show **greyed-out & locked** (visible but not enterable)

## Inside a stage
- **Read** the full lesson part → unlocks the **questions** (sometimes 3 in a row)
- Alternates: read → questions → read → questions ...

## Getting it wrong
- **AI tutor pops up** with **hints only** (never the answer) → **retry**
- **Limited tries** → if you run out → it shows **why** (the explanation) → **re-read** the lesson & redo the stage

## AI Tutor
- **Always available** (tap anytime) **and auto-appears** when you answer wrong
- Personality: **calm, wise teacher**
- Gives **hints only**, never the answer; stays on the current lesson

## Rewards
- Pass → **score + level up** through the 4 ranks: **Ma'lumat → 'Ilm → Fiqh → Fahm**
- Leveling up **unlocks profile decorations** — chosen from a **menu** (emblem skins, name tags, rings, themes)
- **Profile screen** can be opened anytime

## Phase 2 (later)
- Real, safe AI tutor (hints-only, locked to lesson content, no hallucinated verses/rulings)
- User accounts / per-student profiles
